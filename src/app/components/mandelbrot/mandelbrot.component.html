Guide

How to explore

  • Click anywhere on the image to zoom in.
  • Alt+Click zooms out and Ctrl+Click re-centers without changing scale.
  • Iteration count controls detail and render time.
  • Export renders a square high-resolution image; above 9600px can fail.
  • Use custom coloring expressions with the parameter iters.

Mechanics

What's actually happening

Every pixel represents a point c on the complex plane. Starting from z = 0, the same simple formula is applied over and over:

z = z² + c

  • If the size of z ever grows past 2, the point has escaped — it's outside the set, and the pixel is colored by how many iterations that took.
  • If z is still bounded after the iteration cap, the point is treated as part of the Mandelbrot set and takes the color reserved for max iterations.
  • Iteration count trades detail for render time: higher counts resolve finer boundary structure but take longer per pixel.
  • Clicking re-centers the view on that point and rescales the window (zoom in, Alt zooms out, Ctrl only recenters), then every pixel in the new window is recalculated from scratch.

Custom coloring expressions replace the default palette by mapping the escape iteration count (iters) onto each RGB channel, which is why very different expressions can reveal totally different textures from the same underlying math.
